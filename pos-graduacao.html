Pós-Graduação

Análises Clínicas

O curso de Pós-Graduação em Análises Clínicas tem como objetivo capacitar o profissional para atuar em laboratórios de análises clínicas, com ênfase em análises hematológicas, bioquímicas, imunológicas, parasitológicas e microbiológicas, bem como estabelecer o diagnóstico laboratorial, a partir da análise de amostras biológicas, utilizando técnicas e equipamentos específicos.

Banco de Dados

O curso de Pós-Graduação em Banco de Dados tem como objetivo capacitar o profissional para atuar na área de banco de dados, com ênfase em modelagem, administração e desenvolvimento de bancos de dados, bem como na análise de dados e na implantação de sistemas de informação.

Ciência de dados e Big data

O curso de Pós-Graduação em Ciência de Dados e Big Data tem como objetivo capacitar o profissional para atuar na área de ciência de dados e big data, com ênfase em análise de dados, estatística, modelagem, mineração e visualização de dados, bem como na implantação de sistemas de informação.
